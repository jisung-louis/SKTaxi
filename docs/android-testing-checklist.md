## 🔐 권한 테스트

### 1. 위치 권한
- [ ] 앱 실행 시 위치 권한 요청 팝업이 표시됨 (온보딩 화면)
- [ ] 위치 권한 허용 시 지도에 현재 위치가 표시됨
- [ ] 위치 권한 거절 시 적절한 안내 메시지가 표시됨
- [ ] 위치 권한 거절 후 "권한 허용하기" 버튼 클릭 시 Alert가 표시됨
- [ ] 설정에서 위치 권한을 변경한 후 앱으로 돌아왔을 때 정상 동작함

### 2. 알림 권한 (Android 13+)
- [ ] Android 13 이상에서 알림 권한 요청 팝업이 표시됨
- [ ] 알림 권한 허용 시 푸시 알림이 정상적으로 수신됨
- [ ] 알림 권한 거절 시 적절한 안내 메시지가 표시됨
- [ ] Android 12 이하에서는 알림 권한 팝업이 표시되지 않음 (정상)
